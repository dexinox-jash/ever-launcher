## Features

- **Context-aware focus modes** — Morning, Work, Evening, Sleep
- **Mindful gates** — Configurable friction before opening distracting apps
- **Focus score tracking** — 7-day history with insights
- **Focus streaks** — Build and maintain your focus habit
- **Themes** — AMOLED dark theme + light theme
- **Privacy-first** — Zero data collection, zero ads, zero subscriptions

## Installation

1. Download the latest `app-release.apk` from [GitHub Releases](https://github.com/dexinox-jash/ever-launcher/releases/latest).
2. Transfer it to your Android device (Android 8.0+ / API 26+).
3. Install and set EverLauncher as your default launcher.
